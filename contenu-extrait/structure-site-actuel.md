# Structure du Site Actuel - RADC

## Architecture du Site

### Pages Principales
1. **Accueil** (index.html) - Page principale avec carousel hero
2. **À Propos** (about.html) - Informations sur l'organisation
3. **Domaines d'Intervention** (services.html) - Liste des 15 domaines
4. **Équipe** (team.html) - Membres de l'équipe
5. **Actualités** (blog.html) - Articles et nouvelles
6. **Contact** (contact.html) - Formulaire de contact

### Navigation
- Menu principal avec dropdown pour les domaines d'intervention
- Navigation responsive avec menu mobile
- Liens directs vers chaque domaine d'expertise
